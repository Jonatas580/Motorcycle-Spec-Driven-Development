# Motorcycle Spec-Driven Development

## Resumo do Projeto

O projeto é um aplicativo de mobilidade e entregas rápidas via motociclistas, que integra três modalidades de serviço em uma única plataforma:

1.  **Moto Táxi**: Transporte de passageiros dentro da cidade.
2.  **Entrega na Cidade**: Envio de encomendas locais, com cálculo de tarifa urbana.
3.  **Entrega para Outra Cidade**: Envio intermunicipal, com tarifas diferenciadas por distância e pedágios.

## Arquitetura

Este projeto foi estruturado utilizando os princípios da **Arquitetura Hexagonal** para garantir um design limpo, desacoplado e de fácil manutenção. A estrutura de pastas reflete essa escolha, separando claramente o domínio, a aplicação e os adaptadores de tecnologia.
